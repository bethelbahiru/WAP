JQuery Exercise

Question 1. In jQuery terminology, what is the difference between the “jQuery function” and the “jQuery object”?

Answer:

  • A jQuery object is an object returned by the jQuery function.
  • A jQuery function is simply the method to change the state of object.

Question 2. Write jQuery code to find all h1 elements that are children of a div element and make their background color red.

Answer:

  • $("div h1").css("background-color", "red");

Question 3. Use a jQuery method to insert the text "YES!" at the end of the P element ?

Answer:

  • $("p").append("YES!");
