Question 1. Determine what this Javascript code will print out (without running it): Answer:

  • setTimeout(booyah, 2000) -> In this function, booyah is going to be called after 2000 ms.
  • setTimeout(booyah (), 2000) -> In this function, the function call operator () immediately excutes booyah .

    • So, inorder to used setTimeout properly we have to define the function inside without a paranthesis.

Question 3. Write functions booyah1 and booyah2 so that in both cases below, an alert box comes up after 2 seconds that says “BOOYAH!” 

            setTimeout(booyah1, 2000);
            setTimeout(booyah2(), 2000);
Answer: 
            function booyah1() {
                alert('BOOYAH!');
            }
        
            function booyah2() {
                return function () {
                    alert('BOOYAH!');
                }
            }

Question 4. What is "Unobtrusive Javascript"? What is the practical application of Unobtrusive Javascript (and the reasons for using it)?

Answer:

  • It means that the js is placed in a separate file ratherthan embeded into the head of the page.
  • It is useful because sometimes when the js is specified in the head of html, it's fully loaded before the content of the html and it may try to find elements yet to be loaded resulting in a none return.
