Question 2. Define Global Scope and Local Scope in Javascript ?

Answer:

  • Global Scope: Is a scope which is accessible throughout the program. Variables defined here can be used anywhere in the program.
  • Local Scope: Is a scope which is only found in the function it's defined in. Variables defined in this scope are not accessible ouside their definition scope.

Question 4. What is "Unobtrusive Javascript"? What is the practical application of Unobtrusive Javascript (and the reasons for using it)?

Answer:

  • It means that the js is placed in a separate file ratherthan embeded into the head of the page.
