Question 2. What is the difference between a method and function ? Answer:

  • Method: It is a property of an object that contains a function definition. It's called using ObjectName.methodName()
  • Function: It's a block of code which is used to carry out some specific task.

Question 4. What is "Unobtrusive Javascript"? What is the practical application of Unobtrusive Javascript (and the reasons for using it)?

Answer:

  • It means that the js is placed in a separate file ratherthan embeded into the head of the page.
  • It is useful because sometimes when the js is specified in the head of html, it's fully loaded before the content of the html and it may try to find elements yet to be loaded resulting in a none return.
