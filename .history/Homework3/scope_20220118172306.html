Question 2. What is the difference between a method and function ? Answer:

  • Method: It is a property of an object that contains a function definition. It's called using ObjectName.methodName()
  • Function: It's a block of code which is used to carry out some specific task. It's called as functionName()

Question 3. What does 'this' refer to when used in a Java method ? Answer:

  • In java, it's a reference varibale and it refers to the current state of an object.

Question 4. What does 'this' refer to when used in a Javascript method ? Answer:

  • In Javascript, this keyword refers to the enclosing object it belongs to.

Question 5.What does 'this' refer to when used in a JavaScript constructor function? Answer:

  • It is a substitute for the new object to be created. The value of this will become the new object when a new object is created.

Question 6. Assume object x is the prototype for object y in Javascript. Object x has a method f( ) containing keyword 'this'. When f is called by x.f( ), what does 'this' refer to? Answer:

  • this refers to x in this case, because when x inherits the method f() from y, this will become x.
