Question 1. What is the difference between the following 2 statements?

setTimeout(booyah, 2000);

setTimeout(booyah(), 2000);

Answer:
  • setTimeout(booyah, 2000) -> In this function, booyah is going to be called after 2000 ms.
  • setTimeout(booyah (), 2000) -> In this function, booyah is going to be called immediately.

    • So, inorder to used setTimeout properly we have to define the function inside without a paranthesis.
