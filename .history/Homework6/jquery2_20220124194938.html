JQuery 3 Exercise

Question 1. Find the text in the first paragraph (stripping out the html), then set the html of the last paragraph to show it is just text (the red bold is gone).

Answer: 

            $(document).ready(function() {
                var temp = $("p:first").text();
                $("p:last").html(temp);
            });

Question 2. Write jQuery code to create a red background for the level-2 list items.

Answer: 

            $(document).ready(function() {
                $(".level-2").css('background-color', "red");
            });

Question 3. Use a jQuery method to insert the text "YES!" at the end of the P element ?

Answer:

  • $("p").append("YES!");
